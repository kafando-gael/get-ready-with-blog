LesBoutik.com

Plateforme Marketplace Moderne

Présentation de la Plateforme

LesBoutik.com est une application marketplace permettant la mise en relation entre acheteurs, vendeurs et livreurs. La plateforme utilise une architecture microservices pour garantir la scalabilité et les performances.

L'application est développée avec React Native pour le frontend mobile et Node.js pour le backend. La base de données utilisée est PostgreSQL.

Fonctionnalités Principales

  • Authentification sécurisée avec JWT
  • Vérification KYC pour les vendeurs et livreurs
  • Gestion multi-rôles (acheteur, vendeur, livreur)
  • Publication et gestion de produits
  • Système d'offres et de négociations
  • Gestion des commandes et livraisons
  • Paiements intégrés (carte, mobile money, cash)

Architecture Technique

Développée par Gueswende Kafando, Architecte Logiciel

Architecture Microservices

Services indépendants pour l'authentification, le marketplace, la livraison et l'administration. Architecture modulaire et scalable.

Frontend React Native

Application mobile cross-platform pour iOS et Android avec interface native optimisée.

Backend Node.js

API RESTful avec Express.js/Nest.js et base de données PostgreSQL. Architecture conçue pour la performance.

Sécurité

Authentification JWT, vérification KYC, et protection contre les attaques courantes.
